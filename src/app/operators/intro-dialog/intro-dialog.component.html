About

Welcome to RxJs Operators visualizer, here you can visualize RxJs operators in action with a beautiful design about pizza. I really hope you enjoy it because pixel art is not easy :)

Objective:

I've created this web app to make the study of RxJs, used for NgRx (Angular) and react-redux (React/React-Native), more easy. Even for me, developing this visualizer should increase my reactive development skills. I would be very happy if the web community join me in this adventure. Feel free to help the visualizer grow, here you can access the repository on github.

How to use

Streams:

On the left corner of the website you should be able to see a big stream of pizza, the top square represents the last pizza generated by the kitchen. sometimes another stream will show to you for some merge operators.

Operators:

On the bottom-right corner you should see an pizza customer representing the rxjs operator. To subscribe into the stream you should click the button and wait for some interactions

Code:

On the middle of the stream you shpul see the exact code used for the interaction, it will be shown as an image for better experience but you can find the code on the repository as well.
